# bitcrust

Public bug tracker + feature requests for **bitcrust**, the polyphonic
SID-inspired synthesizer from [Anode Labs](https://anode-labs.com).

- **Product page:** https://anode-labs.com/bitcrust/
- **Manual:** https://anode-labs.com/bitcrust/manual/
- **App Store (macOS + iPadOS, universal purchase):**
  https://apps.apple.com/app/bitcrust/id6763228916
- **Gumroad (Windows, Linux):**
  https://store.anode-labs.com/l/bitcrust-win ·
  https://store.anode-labs.com/l/bitcrust-linux

## Reporting bugs

Use the **Issues** tab. The bug-report form walks you through the
useful details (platform, host, version, how to reproduce). A good
report gets a fix faster.

## Requesting features

Same **Issues** tab. Pick the "Feature request" template. Be
specific about the use case — a real scenario is far more useful
than a generic "please add X".

## Discussions

General questions, tips, preset sharing, and "does this count as a
bug?" chatter lives in **Discussions**. Bug reports still go under
Issues.

## Source code

This repo only holds issues, discussions, and project metadata. The
bitcrust source is proprietary and not published. Any references to
internal files or commits in issue comments are for context only;
you can't clone bitcrust from here.
